=== Question 1 ===
==== Define Encapsulation ====
#Answer below

Encapsulation is the method of binding attributes and methods together to create a class, with different access modifiers.

Use cases:
    It is best to use encapsulation when dealing with problems/classes that have sensitive information that should not be made available to other classes
        using access modifiers such as "private" so that certain attributes can not be used by other classes

=== Question 2 ===
==== Give one advantage of Encapsulation ====
#Answer below

The advantage of Encapsulation is the way in which particular access modifiers ensure that certain attributes cannot be used outside of their declared classes

=== Question 3 ===
==== Define Inheritance ====
#Answer below

Inheritance is the usage of attributes and methods of a base (parent) class into a sub (child) class.



=== Question 4 ===
==== What is Method Overriding? ====
#Answer below

Method overriding is a feature in OOP that allows a child class to provide a specific implementation of a method that is already defined in a parent class
